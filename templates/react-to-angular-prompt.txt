CONVERSION GUIDELINES:

JSX to Angular Template:
- className="..." → class="..."
- {condition && <Component />} → @if (condition) { <Component /> }
- {condition ? <A /> : <B />} → @if (condition) { <A /> } @else { <B /> }
- {items.map(item => <Component key={item.id} />)} → @for (item of items; track item.id) { <Component /> }
- style={{...}} → [ngStyle]="{...}" or individual [style.property]="value"
- onClick={() => handler()} → (click)="handler()"
- onChange={(e) => handler(e)} → (change)="handler($event)" or (input)="handler($event)"
- value={state} → [value]="state()" if signal, [value]="state" if property
- disabled={isDisabled} → [disabled]="isDisabled()" if signal, [disabled]="isDisabled" if property
- <> fragments → <ng-container>

React Hooks to Angular:
- useState(initial) → signal(initial)
- useEffect(() => {}, []) → ngOnInit() or effect()

- useEffect(() => { return cleanup }, []) → ngOnDestroy() or effect cleanup
- useCallback(() => {}, []) → regular method
- useMemo(() => value, [dep]) → computed(() => value)
- useRef(null) → viewChild<ElementRef>('refName')
- useContext(Context) → inject(Service)
- Custom hooks → Injectable Service with signals
